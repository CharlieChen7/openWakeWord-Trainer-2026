#### 通过windos下的linux系统训练
先安装wsl
```
wsl --install
```
检查是否安装成功,记得重新打开下cmd
```
wsl -l -v
```

安装python 3.10
```
sudo apt update
sudo apt install python3.10
```

如果版本太新，建议试试旧的wsl
```
wsl --install -d Ubuntu-22.04
```

启动老本版的
```
wsl -d Ubuntu-22.04
```

进入后确认下版本
```
lsb_release -a
```

系统应该默认自带python3.10
```
python3 --version
```
进到自己想要开始项目的文件夹里
克隆项目
```
git clone https://github.com/dscripka/openWakeWord.git
cd openWakeWord
```

补全python组件
```
sudo apt update

sudo apt install software-properties-common

sudo add-apt-repository universe

sudo apt update

sudo apt install \
python3.10 \
python3.10-full \
python3.10-dev \
python3.10-venv
```

安装 venv
```
sudo apt install python3.10-venv
```

创建虚拟环境
```
python3.10 -m venv .venv
source .venv/bin/activate
```

开始安装
```
pip install -e .
```

安装Jupyter
```
pip install notebook ipykernel
```
