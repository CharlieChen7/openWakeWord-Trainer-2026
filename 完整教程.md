#### 通过windos下的linux系统训练
先安装wsl
```
wsl --install
```
检查是否安装成功,记得重新打开下cmd
```
wsl -l -v
```

安装python 3.10
```
sudo apt update
sudo apt install python3.10
```

如果版本太新，建议试试旧的wsl
```
wsl --install -d Ubuntu-22.04
```

启动老本版的
```
wsl -d Ubuntu-22.04
```

进入后确认下版本
```
lsb_release -a
```

系统应该默认自带python3.10
```
python3 --version
```
进到自己想要开始项目的文件夹里
克隆项目
```
git clone https://github.com/dscripka/openWakeWord.git
cd openWakeWord
```

补全python组件
```
sudo apt update

sudo apt install software-properties-common

sudo add-apt-repository universe

sudo apt update

sudo apt install \
python3.10 \
python3.10-full \
python3.10-dev \
python3.10-venv
```

安装 venv
```
sudo apt install python3.10-venv
```

创建虚拟环境
```
python3.10 -m venv .venv
source .venv/bin/activate
```

开始安装
```
pip install -e .
```

安装Jupyter
```
pip install notebook ipykernel
```

安装显卡加速
```
# 1. 卸载可能存在的 CPU 版本，防止冲突
pip uninstall onnxruntime -y

# 2. 安装支持 CUDA 12 的 ONNX Runtime GPU 版本
# 注意：针对 RTX 40 系列显卡，建议安装支持 CUDA 12.x 的版本以获得最佳性能
pip install onnxruntime-gpu --extra-index-url https://aiinfra.pkgs.visualstudio.com/PublicPackages/_packaging/onnxruntime-cuda-12/pypi/simple/

# 3. 安装必要的显卡加速辅助库
pip install nvidia-cudnn-cu12
```

顺便装一下node.js
```
sudo apt update
sudo apt install nodejs npm
```

启动
```
jupyter lab
```
